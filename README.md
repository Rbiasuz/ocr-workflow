# ocr-workflow

Neste projeto, vamos criar um workflow para processamento de imagens

Técnicas para praticar:
 - git
 - docker
 - python
 - ML (OCR)
 - String distance


Objetivo:
 - receber uma imagem
 - coletar o texto da mesma
 - comparar esse texto com uma referência
 - em caso de match -> sinalizar, senão descartar
